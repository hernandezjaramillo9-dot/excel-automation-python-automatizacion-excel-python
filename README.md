# excel-automation-python-automatizacion-excel-python
Personal project for automating expense tracking using Python and Excel. (Proyecto personal para automatizar el seguimiento de gastos usando Python y Excel.)
# Control de Gastos Personales

Este proyecto permite a los usuarios registrar y analizar gastos personales usando Python y Excel.

## Funcionalidades
- Registrar gastos
- Analizar datos con pandas
- Generar resúmenes por categoría

## Tecnologías
- Python
- Pandas
- OpenPyXL

## Uso
Ejecute registrar_gastos.py para añadir gastos.
Ejecute analizar_gastos.py para analizar los datos.
